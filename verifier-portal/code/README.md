# VDSP - Sample Dart Server

This is a sample server built using Dart with the [Shelf](https://pub.dev/packages/shelf) package.  
It demonstrates Verifiable data sharing protocol (VDSP).

---

## Running the Server

```bash
dart bin/server.dart
```

---

---

## Run sample client

```bash
dart example/holder_client.dart
```
