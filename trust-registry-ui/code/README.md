# Trust Registry UI (TRQP)

This is a minimal UI implementation for the [Trust Registry Query Protocol (TRQP)](https://trustoverip.github.io/tswg-trust-registry-protocol/).  
It provides a simple interface to explore organizations, run TRQP queries, and visualize trust registry metadata.
